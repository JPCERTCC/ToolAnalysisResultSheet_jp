- ツール概要

カテゴリ
パスワード、ハッシュ入手
説明
Active Directory から認証情報を取得する。
攻撃時における想定利用例
Golden Ticketの作成やアカウントのパスワード変更に利用される。

標準設定
  • 接続元
    • Prefetch: 実行履歴
追加設定
  • 接続元
    • 監査ポリシー・Sysmon: 実行履歴、実行ファイルからを実行したプロセスによる接続先への通信
  • 接続先
    • 監査ポリシー・Sysmon: Kerberosサービスチケットの操作、特権を持つアカウントによるログオンの監査、Active Directory オブジェクトへの操作
