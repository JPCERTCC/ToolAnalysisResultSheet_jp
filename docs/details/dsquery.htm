- ツール概要

カテゴリ
情報収集
説明
Active Directoryから、ユーザーやグループなどの情報を取得する。
攻撃時における想定利用例
Active Directoryからアカウント情報を収集し、攻撃対象として利用可能なユーザーやホストを選択する。

- 概要

項目 ドメインコントローラー

  • 389/tcp
権限 標準ユーザー (ただし、実行するユーザの権限により取得出来ない情報が存在する)
