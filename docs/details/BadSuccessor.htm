- ツール概要

カテゴリ
権限昇格
説明
Windows Server 2025で導入されたDelegated Managed Service Account（dMSA）機能を悪用することで、攻撃者が任意のユーザーと同等の権限を取得できる。
攻撃時における想定利用例
新規のdMSAアカウントを作成し、CreateChildを保有する任意のアカウントに対して任意の管理アカウントの権限を「継承」する。

  • 接続元
    • 作成されたdMSAアカウントを利用するためにTGSがリクエストされ、イベントID: 4688のログが出力される。ログのCommandLine上に新規に作成されたdMSAアカウント名が記載される
  • 接続先
    • SharpSuccessorの実行によりイベントID: 4741・4722のログが出力される。イベントID: 4741はTargetUserNameとSamAccountNameに、イベントID: 4722はTargetUserNameに新規に作成されたdMSAアカウント名が記載される
