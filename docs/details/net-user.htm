- ツール概要

カテゴリ
アカウント情報の取得
説明
ホストまたはドメイン上の、アカウント情報を取得したり新規に作成したりする。
攻撃時における想定利用例
アカウント情報を収集したり、アカウントを作成して利用したりする。
