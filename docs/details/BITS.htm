- ツール概要

カテゴリ
ファイル共有・転送
説明
バックグラウンドでファイルを送受信する。送受信の際、優先度などを設定することが可能。
攻撃時における想定利用例
他の通信と比較して目立たない程度の帯域で、ファイルを送受信する。

- ログから得られる情報

標準設定
  • 接続元
    • Systemイベントログ: Background Intelligent Transfer Serviceの実行状態が変わることで、判断出来る可能性がある。ただし、BITSが既に動作している場合は判断できない
追加設定
  • 接続元
    • 監査ポリシー・Sysmon: BITSにより作成される一時ファイル"BITS[ランダム数字].tmp"に対する書き込み
    • USNジャーナル: 一時ファイル"BITS[ランダム数字].tmp"のリネーム発生
  • 接続先
    • 監査ポリシー: 共有"\\*\BITS"に対するアクセスの発生

- 実行成功時に確認できる痕跡

  • 接続元
    • 一時ファイル"BITS[ランダム数字].tmp"に対する書き込みが成功している
  • 接続先
    • Microsoft-Windows-Bits-Client/Operationalログにイベント60が記録されており、状態コードが"0x0"となっている
