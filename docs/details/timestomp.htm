- ツール概要

カテゴリ
痕跡の削除
説明
ファイルのタイムスタンプを変更する。
攻撃時における想定利用例
攻撃者が利用したことでタイムスタンプが変更されたファイルのタイムスタンプを戻すことで、ファイルに対してアクセスしたことを隠蔽する。
